## Local Development

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/trainer-portfolio.git
cd trainer-portfolio
```

2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Open http://localhost:5173 in your browser
